# c++ 异常处理



## c++11-noexcept

### 介绍

noexcept  c++11 新增关键字，是运算符。
作用：放在函数定义的后面，说明此函数不向上抛出异常。如果抛出异常，系统会调用terminate终止程序运行。这样保证了，异常不会向上层透出。

经常会用的函数：
析构函数： 如果异常，说明内存泄漏了，这时有异常的话，就需要考虑终止程序了，不要再进行下去了。

移动构造函数： 是 C++11 标准中新加入的一种构造函数，通过这个构造函数，我们可以在对象进行“复制”时，直接“移动”原对象已经分配好的资源，进而省去诸如重新分配内存、再拷贝的过程。
